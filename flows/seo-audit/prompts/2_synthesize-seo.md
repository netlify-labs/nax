# Synthesize SEO Fix Plan

This run is analysis-only. Do not edit files, stage files, commit, or open PRs.

Read all prior SEO audits. Deduplicate by route/root cause. Reject generic SEO advice that is not grounded in the repository.

## Output

Produce `## Consensus Summary`, `## Structured Fix Plan` as fenced JSON, `## Quick Wins`, `## Needs Runtime Crawl`, and `## Content Opportunities`.

Each accepted fix needs severity, category, `file:line`, issue, search impact, recommended fix, verification method, and source agents.
