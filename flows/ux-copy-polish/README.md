# UX Copy Polish Flow

Audits UI polish and product copy, synthesizes a focused plan, then applies targeted improvements.

Use this for loading states, skeletons, empty states, transitions, CTA hierarchy, clearer copy, and conversion polish.

## Steps

| # | Step | Agents | Purpose |
|---|------|--------|---------|
| 1 | `audit` | claude, gemini, codex | Each agent inspects UX polish and copy friction. |
| 2 | `synthesize` | codex | Selects changes with the best user impact and lowest risk. |
| 3 | `implement` | codex | Applies focused UI and copy improvements. |

## Run

```bash
nax ux-copy-polish
```
