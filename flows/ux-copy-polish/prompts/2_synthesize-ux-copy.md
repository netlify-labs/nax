# Synthesize UX Copy Plan

This run is analysis-only. Do not edit files, stage files, commit, or open PRs.

Read all prior UX/copy audits. Deduplicate by user problem and select changes that improve clarity, polish, or conversion without destabilizing core behavior.

## Output

Produce `## UX And Copy State`, `## Structured Polish Plan` as fenced JSON, `## Desktop And Mobile Considerations`, `## Copy Rewrite Notes`, and `## Deferred Backlog`.

Each selected item needs `file:line`, type, current issue, concrete improvement, desktop/mobile impact, verification method, risk, and source agents.
