# Synthesize Documentation Plan

This run is analysis-only. Do not edit files, stage files, commit, or open PRs.

Read all prior documentation audits. Deduplicate by reader problem. Reject style churn unless it improves accuracy, onboarding, or support burden.

## Output

Produce:

1. `## Documentation State`
2. `## Selected Updates` as fenced JSON with target file, gap, evidence, source references, outline, reader, priority, and source agents
3. `## Accuracy Risks`
4. `## Deferred Backlog`
5. `## Implementation Plan`

Do not invent commands, contribution policy, roadmap, badges, or deployment claims.
